添加新单词

这是必填字段
支持JSON数组格式，每个对象包含 word, pos, ph, mean, ex 字段
快速添加格式（用逗号分隔单词）：
success, opportunity, challenge, development
输入用逗号分隔的单词列表，将自动创建为简单单词项

添加新语法点

这是必填字段
支持JSON数组格式，每个对象包含 rule, cat, desc, ex 字段
快速添加格式（用逗号分隔语法规则）：
passive voice, relative clause, modal verbs, phrasal verbs
输入用逗号分隔的语法规则列表，将自动创建为简单语法项

导入文章

支持单个文章JSON文件或包含文章数组的JSON文件
支持的格式示例：
单个文章：
{
  "id": "article1",
  "title": "文章标题",
  "info": { ... },
  "paras": [ ... ]
}
文章数组：
[
  { "id": "article1", "title": "文章1", ... },
  { "id": "article2", "title": "文章2", ... }
]
如果勾选，遇到相同ID的文章将覆盖；如果不勾选，将跳过重复文章
