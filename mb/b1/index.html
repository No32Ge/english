语法点

导入文章

支持单个文章JSON文件或包含文章数组的JSON文件
支持的格式示例：
单个文章：
{
  "id": "article1",
  "title": "文章标题",
  "info": { ... },
  "paras": [ ... ]
}
文章数组：
[
  { "id": "article1", "title": "文章1", ... },
  { "id": "article2", "title": "文章2", ... }
]
如果勾选，遇到相同ID的文章将覆盖；如果不勾选，将跳过重复文章

导出文章

对于多文件导出，将在文件名后添加文章标题

动态导入JS文章库

支持本地文件或远程JS文件，文件应导出文章数组(这种方式已被废弃)

包含150篇进阶英语学习文章
包含80篇商务英语文章
